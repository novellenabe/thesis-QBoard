TO SETUP THE APPLICATION
1. Run venv in the .venv folder
	>> .venv > Scripts > activate
2. CLI: activate

FOR GITHUB BULLSHIT:
1. Clone repo
2. Give access through token
3. Set "Contents" to "Read and Write"
4. When you push it to main repo, it will ask for username and password. <- this only happens when we done any shit yet
5. username: Github username, password: (TOKEN)

FOR VENV:
python -m venv venv


DOWNLOAD:
pip install PyQt5
pip install pyqt5-tools		<-- Optional, but just download anyways
pip install psutil


TO RUN THE APPLICATION
1. python3 shuttleDashboard_interface.py
